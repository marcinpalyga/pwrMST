Preprocessed microarray gene expression datasets. 
They are provided as R data files and contain both the expression matrix and the response variable. 

Leukemia data (2010k), 
Colon data (970k), 
Prostate data (4809k), 
Lymphoma data (1951k), 
SRBCT data (1137k) 
Brain data (1837k). 

For information about the origin and the preprocessing of the datasets, please  read:
Marcel Dettling and Peter B³hlmann,
Supervised Clustering of Genes,
Genome Biology 2002, 3(12) https://www.ncbi.nlm.nih.gov/pmc/articles/PMC151171/pdf/gb-2002-3-12-research0069.pdf
